## 使用说明

1. **何时记录变更**
   - 新增接口/字段
   - 修改接口行为
   - 删除接口/字段
   - 修改数据模型
   - 修改错误码

2. **变更类型说明**
   - `新增`: 完全新的功能
   - `修改`: 已有功能的变更
   - `删除`: 移除功能
   - `废弃`: 标记为即将删除，保留过渡期

3. **向后兼容性**
   - 新增可选字段 = 兼容
   - 新增必填字段 = 不兼容
   - 修改字段类型 = 不兼容
   - 删除字段 = 不兼容
   - 修改接口路径 = 不兼容
   - 修改接口行为 = 视情况而定

4. **变更流程**
   - 变更前评估影响
   - 填写变更记录
   - 通知相关干系人
   - 实施变更
   - 更新相关文档
   - 验证变更结果
