## 设计系统概览

### 项目特征
- **系统类型**: 企业级VM监控管理平台
- **用户群体**: 系统管理员、运维工程师、IT经理、安全工程师
- **使用场景**: 7×24小时运维监控，需要长时间观察数据
- **设备类型**: 主要为桌面端，支持大屏监控
- **国际化支持**: 多语言系统（日文、简体中文、英文）

## 视觉设计提示词

### 整体风格提示词
```
企业级VM监控系统界面设计，采用深色主题为主，专业监控仪表板风格。整体风格简洁、专业、数据导向，参考Prometheus、Grafana等监控系统的设计语言，同时融入现代企业管理系统的视觉特征。

关键特征：
- 深色背景（#0f1419为主色调）减少视觉疲劳
- 数据可视化为主，图表和指标突出显示
- 信息层级清晰，重要数据醒目
- 色彩系统用于状态区分（正常=绿色，警告=橙色，危险=红色）
- 适合长时间观看的护眼设计
```

### 品牌色彩系统
```
主色调系统：
- 背景色：#0f1419（深蓝黑）
- 卡片背景：#1a1f2e（深灰蓝）
- 边框色：#2a3441（浅灰蓝）

状态色彩系统：
- 正常状态：#00d4aa（成功绿）
- 警告状态：#ff9800（警告橙）
- 危险状态：#f44336（危险红）
- 信息状态：#2196f3（信息蓝）

功能色彩系统：
- 主要操作：#4caf50（操作绿）
- 次要操作：#607d8b（中性灰）
- 链接色：#64b5f6（浅蓝）

数据可视化色彩：
- CPU使用率：#2196f3（蓝色系）
- 内存使用率：#4caf50（绿色系）
- 磁盘使用率：#ff9800（橙色系）
- 网络流量：#9c27b0（紫色系）
```

### 排版系统
```
字体层级：
- 主标题：Inter Bold, 24px, #ffffff
- 副标题：Inter SemiBold, 18px, #e0e0e0
- 正文：Inter Regular, 14px, #b0b8c5
- 数据标签：Inter Medium, 12px, #8090a0

行距和间距：

```

### 多语言支持系统
```
支持语言：
- 英文 (en): 默认语言
- 简体中文 (zh-CN): 主要中文用户
- 日文 (ja-JP): 日本市场用户

字体系统（多语言适配）：
英文字体：
- 主字体：Inter（现代无衬线字体）
- 等宽字体：JetBrains Mono（代码/数据显示）

中文字体：
- 简体中文：PingFang SC（macOS）、Microsoft YaHei（Windows）、Source Han Sans CN（跨平台）
- 日文：Hiragino Sans（macOS）、Meiryo（Windows）、Source Han Sans JP（跨平台）

字体回退链：
```css
/* 多语言字体栈 */
font-family: 'Inter', 'PingFang SC', 'Hiragino Sans', 'Microsoft YaHei', 'Meiryo', 'Source Han Sans', sans-serif;
```

语言切换功能：
- 顶部导航栏语言切换器（下拉选择）
- 快捷键：Ctrl+Shift+L 切换语言
- 用户偏好记忆（localStorage存储）
- 登录页面语言选择

文本处理原则：
- 文本容器自适应高度（适应不同语言长度）
- 日文和中文使用更大的行距（1.6-1.8）
- 英文保持标准行距（1.5）
- 数字和单位使用统一格式

日期时间格式：
- 英文：MM/DD/YYYY HH:mm AM/PM
- 中文：YYYY年MM月DD日 HH:mm
- 日文：YYYY年MM月DD日 HH:mm

数字格式：
- 英文：1,234.56（逗号分隔千位）
- 中文：1,234.56（逗号分隔千位）
- 日文：1,234.56（逗号分隔千位）
- 百分比：所有语言使用统一格式（85%）

UI适配考虑：
- 按钮和容器最小宽度考虑文本长度
- 表单标签右对齐（中/日文）vs 左对齐（英文）
- 表格列宽自适应不同语言文本
- 面包屑导航文本截断处理

文本长度规划：
- 英文文本基准：100%长度
- 中文翻译：通常短20-30%
- 日文翻译：通常长10-20%
- 设计时预留30%的文本空间
```

## 页面具体提示词

### 0. 登录页面（移动优先设计）
```
企业级登录页面，移动优先响应式设计：

核心布局：
- 居中的登录卡片（支持触屏操作）
- 大尺寸输入框（适配手指点击）
- 明确的视觉反馈

中央登录区域：
- 系统Logo和"VM监控系统"标题（多语言支持）
- 用户名/邮箱输入框（大尺寸，支持自动聚焦，多语言placeholder）
- 密码输入框（显示/隐藏切换按钮）
- 记住登录状态复选框（大点击区域，多语言标签）
- 忘记密码链接（明显的交互提示，多语言文本）

语言切换器：
- 登录框右上角语言切换下拉菜单
- 显示当前语言图标和名称
- 支持快速切换语言
- 语言切换后立即更新界面文本

多语言文本示例：
- 英文："VM Monitoring System" | "Username" | "Password" | "Remember me" | "Forgot password?"
- 中文："VM监控系统" | "用户名" | "密码" | "记住我" | "忘记密码？"
- 日文："VM監視システム" | "ユーザー名" | "パスワード" | "ログイン状態を保持" | "パスワードを忘れた場合"

操作区域：
- 登录按钮（主要操作，带加载状态）
- 错误提示区域（明显但不过度干扰）
- 账户状态提示（锁定、过期等）

背景设计：
- 深色科技感背景（#0f1419）
- 抽象的VM或监控相关图形元素
- 版权信息和系统版本（底部）

移动端优化：
- 键盘弹出时页面自动上移
- 触摸友好的按钮尺寸（最小44px）
- 支持生物识别登录预留位置
- 手势操作（左滑显示更多信息）

桌面端增强：
- 更大的屏幕空间利用
- 品牌宣传内容区域
- 安全提示信息面板
- 快捷键支持（Enter登录、Tab切换）

响应式断点：
- 手机：320px-768px（卡片全宽）
- 平板：768px-1024px（卡片最大480px宽）
- 桌面：1024px+（卡片固定500px宽）

错误处理：
- 登录失败时的震动反馈
- 账户锁定时的倒计时显示
- 密码重置流程入口
- 网络异常的重试机制

无障碍设计：
- 完整的表单标签关联
- 屏幕阅读器支持
- 高对比度模式适配
- 键盘完整导航
```

### 1. 主仪表板页面（健康+故障双模式）
```
主仪表板页面，智能健康/故障双模式切换：

顶部导航栏：
- 左侧：系统Logo + "VM监控系统"文字（多语言）
- 中部：主导航菜单（多语言：Dashboard/仪表板/ダッシュボード、VM List/VM列表/VM一覧、History/历史数据/履歴データ、Alerts/告警管理/アラート管理、Users/用户管理/ユーザー管理）
- 右侧：用户信息、通知铃铛（显示未读告警数量）、语言切换器、模式切换按钮

语言切换功能：
- 下拉菜单显示三种语言选项
- 实时切换界面语言
- 保持用户当前页面状态
- 语言偏好自动保存

**正常模式（默认显示）**：
顶部区域 - 系统健康总览：
- 系统健康评分大卡片：环形进度图 + 评分数字
- 评分说明：基于在线率、性能指标、告警数量计算
- 评分历史趋势小图表（最近7天）

中上部 - 核心指标概览（4列网格）：
- CPU使用率卡片：当前百分比 + 趋势箭头 + 迷你折线图
- 内存使用率卡片：已用/总量 + 使用率 + 趋势图
- 磁盘使用率卡片：整体使用率 + 最满的磁盘警告
- 网络流量卡片：实时带宽 + 流量趋势图

下方内容区域（2列布局）：
左下方 - VM状态分布：
- VM状态饼图：在线/离线/警告/严重
- 按部门/集群分组的VM统计
- 快速筛选标签页

右下方 - 最新告警摘要：
- 最多显示5条最新告警
- 每条显示：时间、VM名称、告警类型、严重程度
- "查看全部"链接到告警管理

**故障模式（检测到严重告警时自动切换）**：
顶部紧急通知横幅：
- 红色闪烁背景，告警数量和严重程度
- "切换回正常模式"按钮

主区域 - 问题VM列表：
- 按严重程度排序的问题VM卡片
- 每个卡片显示：VM名称、IP地址、问题类型、紧急程度、处理建议
- 快速操作按钮：查看详情、重启、联系负责人
- 状态指示灯：红色闪烁表示严重问题

右侧快速操作面板：
- 批量操作按钮：批量重启、批量通知
- 问题快速定位：按集群、按问题类型筛选
- 紧急联系人信息

**模式切换逻辑**：
- 自动切换：检测到Critical级别告警或超过5个High级别告警
- 手动切换：用户可通过顶部按钮手动切换
- 动画过渡：模式切换时有平滑的淡入淡出效果
- 状态保持：用户手动切换后记住偏好设置

**移动端适配**：
- 导航栏折叠为汉堡菜单
- 系统健康评分移至顶部卡片
- 核心指标改为横向滚动卡片
- VM状态使用列表视图
- 问题VM使用垂直卡片列表，支持滑动操作
- 底部固定快速操作栏

**交互特性**：
- 实时数据更新：每30秒刷新一次
- 悬停效果：卡片悬停时显示详细信息
- 快捷操作：Ctrl+R刷新，Ctrl+A查看所有告警
- 键盘导航：Tab键顺序导航，Enter键确认操作
```

### 5. 用户管理页面（简化权限展示）
```
用户管理界面，简化权限展示理念，隐藏继承复杂性：

**用户列表标签页**：
顶部操作栏：
- 搜索框：按姓名、邮箱、部门搜索
- 筛选器：按角色、状态、最后登录时间筛选
- 批量操作：批量启用/禁用、批量角色分配
- 新增用户按钮

用户列表表格：
- 列：姓名、邮箱、角色、状态、最后登录时间、操作
- 状态指示：正常（绿色）、待审核（橙色）、已禁用（红色）、已过期（灰色）
- 操作按钮：编辑、重置密码、启用/禁用、查看权限

用户编辑模态框：
- 基本信息：姓名、邮箱、部门、电话
- 角色分配：多选角色选择器，显示角色描述
- 状态管理：启用/禁用、设置过期时间
- 安全设置：强制下次修改密码、启用MFA（预留）

**角色管理标签页**：
角色列表：
- 树形结构展示角色层级关系
- 每个角色显示：名称、描述、用户数量、创建时间
- 操作：编辑、删除、查看权限

角色编辑模态框：
- 基本信息：角色名称、描述、父角色选择
- 权限设置：功能级权限选择器
- 用户列表：当前角色下的所有用户
- 实时权限预览：显示最终有效权限（合并继承权限）

**权限矩阵标签页**：
权限矩阵表格：
- 行：角色名称（按层级排序）
- 列：功能模块（监控数据查看、告警管理、系统配置、用户管理）
- 单元格：权限级别标签（无权限、只读、编辑、管理）
- 支持批量权限设置：选择多行设置相同权限

权限详情面板：
- 点击单元格显示该角色在此模块的详细权限
- 显示权限来源：继承权限 vs 直接授予
- 权限级别说明：每个级别允许的具体操作

**权限管理逻辑**：

// LanguageSwitcher 组件
import React from 'react';
import { useTranslation } from 'react-i18next';

const LanguageSwitcher: React.FC = () => {
  const { i18n } = useTranslation();

  const languages = [
    { code: 'en', name: 'English', flag: '🇺🇸' },
    { code: 'zh-CN', name: '简体中文', flag: '🇨🇳' },
    { code: 'ja-JP', name: '日本語', flag: '🇯🇵' }
  ];

  return (
    <select 
      value={i18n.language} 
      onChange={(e) => i18n.changeLanguage(e.target.value)}
      className="language-selector"
    >
      {languages.map(lang => (
        <option key={lang.code} value={lang.code}>
          {lang.flag} {lang.name}
        </option>
      ))}
    </select>
  );
};
